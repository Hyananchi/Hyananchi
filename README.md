Обо мне
-------

Я начинающий python разработчик, свой путь в it я начинал с простых bat скриптов и самых примитивных html страничек. Спустя пару лет я решил для себя что хочу связать свою жизнь с кодингом и с 16 лет начал писать свои первые "программы" на языке C. К сожалению сишка мне не особо зашала из-за своего не совсем удобного синтаксиса и начал пробовать искать другие языки и одим из них оказался Python. Спустя почти 4 года я без проблем пишу APIшки для back-end и даже балуюсь с нейросетями!
